# Packet Tracer - Configure Initial Switch Settings

## Lab Overview
This lab demonstrates basic switch configuration on Cisco Catalyst 2960 switches using Packet Tracer. Two switches (S1 and S2) are configured with essential security and administrative parameters.

## What Was Accomplished

### Switch Configuration Tasks
- **Device Naming**: Assigned hostnames to switches (S1 and S2)
- **Console Access Security**: Protected console ports with password authentication
- **Privileged Mode Security**: Configured enable passwords and enable secret passwords
- **Password Encryption**: Applied service password encryption to secure all plain-text passwords
- **MOTD Banner**: Added message-of-the-day banners to warn unauthorized users
- **Configuration Persistence**: Saved running configurations to NVRAM (startup-config)

### Skills Demonstrated
- Navigation between CLI modes (User EXEC → Privileged EXEC → Configuration mode)
- Securing physical and remote access to network devices
- Password encryption and security best practices
- Configuration backup and recovery procedures
- Multi-device configuration consistency

## Lab Details
- **Topology**: Two Cisco Catalyst 2960 switches (S1, S2)
- **IOS Version**: 12.2 (can be updated to 15.0)
- **Key Passwords**: Console and enable secret passwords configured for authentication
- **Configuration**: Saved to NVRAM for persistence
